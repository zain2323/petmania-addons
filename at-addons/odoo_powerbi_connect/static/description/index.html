Microsoft Power Bi Integration with your Odoo.

Odoo Power Bi Connector allows you to integrate your Odoo with Microsoft Power BI business intelligence tool. It supports multi workspace and helps you to create dynamic server actions for any model.
You can also configure cron to schedule export of data for any model/table. You can easily export or import data to track and analyze the growth of your business.
It helps you to make better decisions for your business. It has a well-defined dashboard which makes data clear to you.

Detailed Features List

Below is the list of features of Odoo Power BI Connector.

Create or Export Datasets

  • You can create datasets with one or more tables.
  • Moreover, you can also export datasets to powerbi.

Multi Workspace Support

  • The module supports multiple workspaces.
  • You can import workspaces from Power BI too to Odoo.
  • you can also create workspaces at Odoo and then export.

Filters for the Ease to Search Data

  • You can define filters to distinguish the data from tables.
  • For example, you can filter products from different groups, countries, etc. which depends on the selected model fields.

Ease to export data from Odoo

  • You can export data of any odoo model by creating tables.
  • You can configure cron to schedule export of data for any model/table.

Managing Dynamic Server Actions

  • You can create dynamic server actions for any model from datasets.
  • It helps you to export the data from the report of that model to Power BI tool.

Track and Analyze data from the dashboard

  • Import reports and dashboards from powerbi.
  • You can also visualize these reports and dashboards from Odoo.
