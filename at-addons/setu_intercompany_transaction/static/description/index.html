In any industry today inventory movements between companies or between warehouses are common operations. It’s really hard to track each movement and its corresponding records. Especially when there are more companies doing transactions with each other, it becomes painful to manage track records for companies.

We have developed a solution called “Inter Company Transaction” which helps to manage all transactions between companies in an automated way. All required documents will be created automatically by just a single click. Users can get all historical transactions (deliveries, receipts, invoices, vendor bills) very easily.


  • Special feature ( Stock + Inter warehouse transfer)

  • Pick the available stock first and the rest will be picked up from other warehouses of the same company. When a sales order gets created, it will reserve available stock and for the rest, it will create inter warehouse transaction records between the sales order's warehouse and the other warehouses where the stock is available. inter warehouse transaction will be created based on the inter warehouse channels configured in the system.

    • Key Features


  • Inter Company Channels
  • Each company can configure its source from where they will purchase as and when needed. More than one channel can be configured and the system will pick the right one according to priority.
    •
  • Inter Warehouse Channels
  • Each company can configure its source warehouses from where they will replenish stock as and when needed. More than one channel can be configured and the system will pick the right one according to priority.
    •
  • Inter company transfer
  • It allows stock movements between companies by creating sales orders in the fulfiller company and purchase orders in the requestor company. Along with stock movement it will take care about accounting documents.
    •
  • Inter warehouse transfer
  • It allows stock movements between warehouses of the same company by creating two step pickings (Source -> Transit -> Destination) or direct transfer (Source -> Destination) in a single step (based on configurations).
    •
  • Auto create Inter Warehouse Transaction
  • Companies can configure when an inter warehouse transaction needs to be created automatically (create Inter warehouse when receive sales order, when there isn’t enough stock for all products, when there isn’t enough stock for any single product, manually)
    •
  • Auto create Inter Company Transaction
  • Companies can configure when an inter company transaction needs to be created automatically (create Inter company when receive sales order, when there isn’t enough stock for all products, when there isn’t enough stock for any single product, manually)
    •
  • Direct delivery to customer
  • It’s similar to dropship, if the requestor company wants to deliver orders to customers directly from the fulfiller company then the fulfiller company will deliver orders directly to customers instead of the requestor company's warehouse.
    •
  • Automatic documents processing
  • There is an option in the channels / inter company transaction which will process all the documents according to the configurations. Based on the configuration system will create / validate all the documents.
  • Easy return management
  • Helps to manage inventory return process, there is an easy interface from where users can return stock to companies or warehouses.
    •
  • Easy import feature
  • Helps to create inter company and inter warehouse transactions record by importing simple csv files.
    •
