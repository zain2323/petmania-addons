Odoo Advanced Search | List view Manager

The module allows you to quickly search records from the list view.

Features

Listview Manager

The quick search section is available over all list/tree views.

Quick Search

As soon as you hit Enter in any quick search field or focus away, your search takes effect immediately.

Multiple search

Multiple search queries can be performed over many2one fields. (Ex.:Customer is Gemini Furniture or Lumber Inc)

Date Range

Fields of type date and datetime can be filtered by date ranges.

Group By

By working with the group, filters can achieve their goals.

User Preferences

Date filter can be set as date range or normal according to the user's preferences.
