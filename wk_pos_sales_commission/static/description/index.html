Detailed Features List

Integrated POS SalesPerson Commission Solution in Odoo

  • Integrated POS SalesPerson Commission Solution in Odoo
  • Map out commission for POS products, POS users, POS shops, etc.
  • Create commission invoices and reports- all in Odoo.

Define POS Sales Commission Rules using configurations

  • Use Commission configuration to set rules for earning commissions.
  • Set commission on products or Total POS Cart amount.
  • Set the time in Days or hours during which the commission configuration is valid.

Set Commission rules for Products Or Cart amount

  • For Products - Choose All products or Product categories or specific product(s).
  • For Total Cart Amount - select different amount ranges and their respective earned commissions.

Choose POS Sales Commission Types - percent or fixed

  • POS Sales Commission can be on the basis of fixed amount or percentage.

Create Multiple Commission Rules Configurations

  • You can create multiple POS commission configurations in Odoo.
  • Each commission can have its own rules and commission types.
  • Each configuration works independent from another.

Assign different Configurations at different levels - User, Shop, Global

  • Option to use different Commission Configurations ate different level.
  • You can set separate configuration for POS User, Employee and POS Shop.
  • Or, you can use one global/default configuration for all of them.
  • Hierarchy of levels in decreasing preference : User >> POS SHOP >> Global/Default.

Download POS Salesperson Commission Reports

  • Generate POS Sales Commission PDF reports on User or Employee Basis.
  • Allows to select the time period for Each commission report.

Maintain POS Commission Records in Odoo

  • The module records every credited POS commission in backend.
  • You can choose to create separate commission record for every product in an order.
  • Or Create single record with total commission of entire POS order and details.
  • Backend View
  • POS Session View

Commission can be calculated in two ways : Fixed or Percentage.Start & End Date can be set for the Commission.

  • Commission can be applied based on All Products, Categories or Products

  • Multiple Commission rules can be Created for differenct Cart Amount.
